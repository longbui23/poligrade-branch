# PoliGrade

Democracy Powered by Data, Not Drama

A political accountability platform that helps voters make informed decisions through data-driven evaluations of politicians and policy-based party alignments.

## Features

- **Politician Grades**: Search and filter 589 elected officials by name, district, office, and political alignment
- **Voter Alignment Quiz**: 27-question quiz across 9 policy areas to discover your political alignment
- **Six Policy Parties**: Progressive, Liberal, Centrist, Moderate, Conservative, and Nationalist classifications
- **Mobile-First Design**: Fully responsive with optimized mobile experience

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- NextUI

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the application.

## Build

```bash
npm run build
npm start
```
